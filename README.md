# PrimeSlidingMenu

ECSlidingViewController 2 integration for MotionPrime.

## Installation

Add this line to your application's Gemfile:

    gem 'prime_sliding_menu'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install prime_sliding_menu
